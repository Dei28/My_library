# 📚 Книжная полка (MyLibrary)

Веб-приложение на Django для каталогизации книг. Пользователи могут просматривать книги, добавлять их в избранное, оставлять отзывы (в разработке) и переходить к чтению через Google Книги. Администратор управляет каталогом через удобную админ-панель.

## ✨ Возможности

- Регистрация и аутентификация пользователей
- Просмотр списка книг с обложками и кратким описанием
- Детальная страница книги с полной информацией, кнопкой "В избранное" и автоматическим поиском в Google Книгах
- Страница автора со списком его книг
- Личный кабинет с избранными книгами
- Админ-панель для управления книгами, авторами, жанрами и пользователями
- Адаптивный дизайн на Bootstrap 5

## 🛠 Технологии

- Python 3.14
- Django 6.0.3
- Bootstrap 5
- SQLite (по умолчанию, можно переключить на PostgreSQL)
 SQLite (с возможностью переключения на PostgreSQL)
Что это?
SQLite — это легковесная встраиваемая реляционная база данных. Данные хранятся в одном файле (обычно db.sqlite3). Не требует отдельного сервера, настройки, паролей.
- Pillow (для работы с изображениями)
Pillow (для работы с изображениями)
Что это?
Pillow — библиотека Python для обработки изображений. Она позволяет открывать, редактировать и сохранять картинки в различных форматах (JPEG, PNG, GIF и др.).

Зачем в проекте?
В моделях Book и CustomUser есть поля ImageField (cover и avatar). 

## 🚀 Установка и запуск

1. **Клонируйте репозиторий:**
   git clone https://github.com/Dei28/My_library.git
   cd My_library


# Создание и активация виртуального окружение
python -m venv venv
venv/Scripts/activate      

## установка зависимостей 
pip install -r requirements.txt

## применение миграции 
python manage.py migrate

## создание суперпользователя 
python manage.py createsuperuser

## запуск сервера 
python manage.py runserver

# 🧪 Запуск тестов и измерение покрытия
## Запуск всех тестов

## Отчёт в консоли

## HTML-отчёт (открыть htmlcov/index.html)
